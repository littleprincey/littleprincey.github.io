what are control flow and loops

you’ve opted to take a ride on some awful rollercoaster at your cousin’s birthday party at disney land. you board after waiting in a long, sweaty line and the carriage takes off along the tracks. yes its awful, yes there is screaming and terror and all the jolly things but that carriage is on those tracks and set to go in only one direction, on a predetermined path and you will inevitably face the dreaded corkscrew. this is like control flow in code. the computer executes the code in a systematic way, like a carriage on tracks, until it is done. the path travelled is determined by the tracks which have already been built and thats the only way to go. you cant eject from the seat or suddenly switch to the teacups way half way through. but say you’re one of those interesting people that loves being terrified and feeling nauseous. say you want to keep going and not just do the one, meagre circuit of pain and terror. a loop in coding allows code to repeat until a specific condition is met, interrupting the control flow. so we could write a piece of code that says- while(youLovePain === true && youWantToStop === false) {rideOn();} . this is a loop that, due to the syntax, the code will execute over and over until one or both of these conditions is not met. so you can just go and go and go on that sick ride. thanks to loops.

functions

functions are blocks of reusable code that perform a specific task. parameters can be passed through the function that it can check data against and perform certain actions based on those parameters and data, returning a result. it means that instead of writing something over and over and over again, we can write a function that will write this thing a certain amount of times or until a specific condition is met. functions can be ‘called’ and then their programmed function will execute with whatever inputs the programmer has defined, and presents the output. they are an essential tool to writing eloquent, condensed code that can perform complex tasks in less code.
